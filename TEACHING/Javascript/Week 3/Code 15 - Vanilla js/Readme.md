## Steps to setup:

**Step 1:**
Install nodejs - version 22

**Step 2:**
Setup a empty `package.json` file. Run the command:
```bash
npm create vite@latest <app name> --template vanilla
```

**Step 3:**
Install the project
```bash
npm i
```

**Step 4:**
Run the project
```bash
npm run dev
```
Go to http://localhost:5173/ in browser
